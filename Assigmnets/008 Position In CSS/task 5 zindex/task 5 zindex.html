Z-Index Stacking Order Example

This example demonstrates how the CSS z-index property controls the stacking order of overlapping elements.

Explanation of Boxes

  • Box 1 (Red): Positioned at (50px, 50px) with a z-index of 1. It is behind the blue box.
  • Box 2 (Blue): Positioned at (70px, 70px) with a z-index of 2. It overlaps Box 1 and appears in front of it.
  • Box 3 (Green): Positioned at (90px, 90px) with a z-index of 0. It is behind both Box 1 and Box 2.

Understanding Stacking Context

The z-index property only affects positioned elements. The higher the z-index value, the closer the element is to the viewer, allowing it to cover elements with lower z-index values.

In this example, even though the boxes overlap, their stacking order is determined by the z-index values assigned to them. Box 2, with the highest z-index, appears on top, while Box 3, with the lowest z-index, is hidden behind the others.
